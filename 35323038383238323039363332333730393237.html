Bem-vindo à Code Barber

Na Code Barber, combinamos a arte clássica da barbearia com as tendências modernas para criar um estilo que é unicamente seu. Nosso ambiente é projetado para oferecer uma experiência de luxo e relaxamento, onde cada corte, barba ou tratamento é executado com precisão e paixão.

Localizada no coração de Belo Horizonte, somos o refúgio perfeito para o homem que busca excelência e um cuidado impecável. Venha descobrir a diferença de um serviço verdadeiramente especializado.
